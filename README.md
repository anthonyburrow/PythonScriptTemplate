# Python Script Template

A minimal Python project template for quick script development.

## Quick Start

```bash
make all
```

## Available Commands

- `make all` - Create venv and install dependencies (recommended first step)
- `make venv` - Create virtual environment only
- `make install` - Install project dependencies
- `make clean` - Remove build artifacts and virtual environment
- `make help` - Show available commands

## Dependencies

Quick spin-up dependencies defined in `pyproject.toml`:
- numpy
- scipy
- scikit-learn
- matplotlib
